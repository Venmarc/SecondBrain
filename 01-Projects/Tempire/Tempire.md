# Tempire — Project Overview

> "Build your empire, one template at a time."
> Live at: https://tempire.xyz/

## What It Is
Premium niche marketplace for vetted, production-ready digital assets — templates, UI kits, AI prompts, Figma assets, ebooks. Solves the blank page problem for developers, designers, and founders.

## What Makes It Different
- Lego-style approach: architectural bedrock (Next.js, Tailwind, Supabase, TypeScript) so buyers focus on unique logic
- 20-point quality check on every asset: code performance, design accessibility, documentation
- Lifetime ownership — pay once, own forever including future updates
- "48 Laws of Web Design" aesthetic standard across all assets

## Tech Stack
- Next.js 16+ App Router + TypeScript (strict)
- Tailwind CSS 4 + shadcn/ui
- Clerk (auth + RBAC — buyer/seller roles)
- Supabase (Postgres + Realtime + Storage) — server-side only
- TanStack Query (client-side state + caching)
- Zod (form validation)
- @vercel/og (dynamic OG images)
- Deployed on Vercel

## Current State
89% complete by Victor's standards. Core flows done. UI/UX polish is the remaining 10% — and the hardest part.
